📊 BLOCKED CONTROL

Ferramenta desenvolvida para controle e visualização de inventário bloqueado, transformando registros operacionais e analises em informação clara para acompanhamento e tomada de decisão.

🚀 Funcionalidades

Registro diário de caixas bloqueadas /
Visualização em gráfico por data /
KPI automático do último inventário /
Edição e exclusão de registros /
Exportação de dados em Excel /
Exportação do gráfico em imagem (PNG)

💡 Problema Resolvido

Em muitos processos logísticos, o controle de inventário bloqueado não tem facil visibilidade.

Este projeto transforma dados operacionais em um dashboard simples, visual , eficiente, e facilitando a tomada de decisão.

🛠️ Tecnologias Utilizadas

HTML
CSS
JavaScript
Chart.js
